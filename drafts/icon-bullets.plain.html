Icon bullet list demo

Lists below use :icon-search: and :icon-check: at the start of each item. The list-style bullet is replaced by the icon; wrapped text aligns with the first line.

  • :icon-search: Here is LI number one
  • :icon-search: Second item is here
  • :icon-check: Task one
  • :icon-check: Task two with longer text so we can see that when this line wraps to a second line it should align with the start of the first line and not under the icon.
